# Sum Series Logarithms - Вычисление логарифмического ряда

## Описание программы

Программа вычисляет значение функции `ln(x + sqrt(x² + 1))` с помощью разложения в степенной ряд (ряд Тейлора). Реализовано три способа остановки вычислений: по заданному количеству членов ряда, по достижению заданной точности и автоматическое вычисление с повышенной точностью.

## Математическая основа

Программа использует разложение в ряд:
```
ln(x + √(x² + 1)) = x - (1/2) * (x³/3) + (1*3)/(2*4) * (x⁵/5) - ... 
```
где радиус сходимости R = 1.

## Использование

### Входные данные

Программа запрашивает три параметра:

1. **x** - аргумент функции (должен быть в диапазоне от -1 до 1 включительно)
2. **Число членов ряда (N)** - максимальное количество итераций для вычисления
3. **Точность (Eps)** - требуемая точность вычислений (должна быть ≤ 1)

### Выходные данные

Программа выводит:

1. Результат вычисления по ряду при достижении заданного числа членов N
2. Результат при достижении заданной точности Eps
3. Результат при достижении точности Eps/10 (автоматическое уточнение)
4. Точное значение, вычисленное с помощью стандартной математической библиотеки

### Пример работы

```
x (от: -1, до: 1) -> 0.5
Число членов ряда ->  10
Точность ->  0.0001

sum=0.4812118 N=10
sum= 0.4812118 K=5
sum= 0.4812118 K=5
Результат формулы: 0.481211825
```

## Особенности реализации

- **Контроль входных данных**: Проверка корректности введенных значений
- **Тройной критерий остановки**: 
  - По количеству итераций
  - По заданной точности
  - По автоматически увеличенной точности
- **Эффективное вычисление**: Используется рекуррентная формула для вычисления членов ряда
- **Сравнение с эталоном**: Вывод значения, вычисленного стандартными средствами .NET

## Формулы вычисления

Текущий член ряда вычисляется по рекуррентной формуле:
```
currentElement = -currentElement * x2 * (2 * k + 1)² / ((2 * k + 2) * (2 * k + 3));
```
где k - номер текущей итерации.

## Ограничения

- Аргумент x должен находиться в диапазоне [-1, 1] для обеспечения сходимости ряда
- Точность должна быть ≤ 1
- Программа использует тип double для вычислений (точность ~15 знаков)

## Применение

Данная программа может быть использована для:
- Изучения методов численного анализа
- Демонстрации сходимости степенных рядов
- Сравнения эффективности различных критериев остановки
- Проверки точности численных методов
